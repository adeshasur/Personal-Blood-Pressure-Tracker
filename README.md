# Personal Blood Pressure Tracker

A modern, privacy-focused Blood Pressure Tracker built with React, Tailwind CSS, and Lucide Icons. Designed for simplicity and hosted entirely on the client-side.

![Blood Pressure Tracker Preview](https://via.placeholder.com/800x450?text=BP+Tracker+Dashboard)

## Features

- **Dashboard**: Get an immediate summary of today's readings and trends.
- **Log Readings**: Easily record your systolic, diastolic, and pulse.
- **Visual Trends**: Interactive charts showing your blood pressure and pulse history over time.
- **Privacy First**: All data is stored locally in your browser's `localStorage`. No data ever leaves your device.
- **Modern UI**: Clean, responsive design with glassmorphism aesthetics.

## Deployment

The application is optimized for hosting on **Vercel**. Since it is a pure frontend application, it can be deployed with zero configuration.

### Local Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```

## Built With

- **React**: Frontend framework
- **Tailwind CSS**: Styling and layout
- **Lucide React**: Icon system
- **Recharts**: Data visualization
- **LocalStorage**: Client-side data persistence
